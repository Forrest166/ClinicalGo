Step2A / Step2C sample-size policy:

1. Prefer an arm-level sample size when the extracted row clearly corresponds to one arm and that arm count is explicit.
2. Do not infer arm counts from total study sample size.
3. If only a total sample size is available and the row is not clearly arm-specific, keep the total.
4. If the row is arm-specific but only the total is known, leave the row-level sample size blank rather than forcing a split.
